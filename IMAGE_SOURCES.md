## 如何下載和上傳圖片

1. 訪問上述每個 URL
2. 2. 下載圖片檔案
   3. 3. 將檔名按照對應規則重命名（例如：`2025/07/fb.png` → `2025_07_fb.png`）
      4. 4. 上傳至 GitHub 的 `img/` 資料夾
        
         5. ## 檔案結構
        
         6. ```
            myblog/
            ├── article/
            │   └── 2025-07-04-heat-point-keeper.md
            ├── img/
            │   ├── 2025_07_fb.png
            │   ├── 2025_07_04_5_45_15.png
            │   ├── 2025_07_04_5_45_59.png
            │   ├── 2025_07_04_5_46_31.png
            │   └── 2025_07_04_6_20_10_2.gif
            └── ...
            ```

            ## 備註

            文章 Markdown 檔案中的圖片路徑已設定為相對路徑 `../img/filename`，因此只需將圖片上傳到 `img` 資料夾即可自動連接。
